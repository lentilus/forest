Terminology. Reasonable class of morphisms [00BN]

    We say a class of morphisms of schemes is reasonable if it fullfills the following properties.

    1. All isomorphisms are in the class.
    2. The class is closed under composition.
    3. The class is closed under base change: if is in the class, and is in the class, then the induced map is also in the class.
    4. (The class is local on the target)
