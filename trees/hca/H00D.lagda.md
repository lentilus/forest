# What we need
- an _agda_ installation
- a nice text editor (_Neovim_ + _Cornelis_)
- literate programming support

# Working with Cornelis

We will assume two Types `A` and `B`.
```agda
data A : Type where
  a1 : A
  a2 : A

postulate
  B : Type

```
Load agda files with `:CornelisLoad`

Goals `{!!}`, `:CornelisTypeContextInfer`
```agda
someFunc : A -> B
someFunc a = {!!}
```

Split cases with `:CornelisLoad`.
```agda
-- switch : A -> A
-- switch a1 = a2
-- switch a2 = a1

switch : A -> A
switch a = {!!}
```

What is `switch a1`? Normalize expressions with `:CornelisNormalize`
